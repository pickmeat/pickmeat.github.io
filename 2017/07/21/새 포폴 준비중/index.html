    Icis 맛있는 행복


    전에 만들었던 사이트랑 크게 다르지 않다. 다만 디자인과 반응형에 신경 많이 씀.
    회사 다닌 후에 실력도 많이 올라서 이젠 훨씬 빨리 만든다. 이 기세로 사이트 찍어내야지ㅋㅋ

    라이브러리는 부트스트랩과 sematic UI 이용. sematic UI에 이쁜거 많아서 좋긴한데 대부분 부트스트랩이 대체 가능하고 활용하기도 더 쉽다. 그래도 기왕 만든거 그거 위주로 만드는 중.

    지금 웬만한 기능은 다 구현했고(회원가입, 게시판, 로그인, 덧글 등등…) 관리자 페이지를 신경 써서 만드는 중이다. 전에 만들었던 홈페이지는 따로 관리자 페이지를 안 만들어서 매번 DB 켜느라 귀찮귀찮


    관리자 페이지는 아코디언 메뉴로 만들었다.
